template<auto X>
class xu::UniqueSlot< X >

RAII-based slot object which will automatically call Connect and Disconnect on construction and destruction, respectively. This prevents dangling slots since Signal is not aware that a member function pointer instance is no longer valid and will wrongly continue to blindly invoke it.
